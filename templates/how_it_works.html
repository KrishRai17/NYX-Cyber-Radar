HOW IT WORKS

Phishing Detection Analysis & Visualization Logic

01

Heuristics Analysis

When an endpoint is submitted, the heuristic scanner parses the target URL string for potential threat markers. It scores the risk level according to four metrics:

HTTPS check Keyword entropy String length limit Direct IP targeting
02

IP-API Geolocation Trace

The server resolves the target domain name into its underlying IP address using DNS lookup. It queries the IP-API database to pinpoint the server's geographical coordinates (latitude and longitude), registration country, and managing Organization/ISP.

DNS Resolution IP Mapping ISP Telemetry Lat / Lon Coordinates
03

3D Cartesian Projection

The latitude and longitude coordinates are translated mathematically to a 3D space vector using trigonometric coordinate transformation. The frontend draws an animated cyber-globe representing earth, calculates bezier arcs from the client location to the target node, and sweeps a laser comet particle along the connection path to represent connection data flow.

Spherical coordinates Bezier curves Interactive orbit Laser beacons
